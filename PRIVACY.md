# 隱私政策

最後更新日期：2025 年 1 月 6 日

「星夜信封」（以下簡稱「本服務」）重視您的隱私。本政策說明我們如何收集、使用和保護您的資料。

## 1. 資料收集

### 我們收集的資料

| 資料類型 | 說明 | 用途 |
|---------|------|------|
| 匿名識別碼 | 自動產生的 UUID | 識別您的帳號 |
| 暱稱 | 您自行設定（選填） | 顯示在瓶子上 |
| 縣市 | 您自行設定（選填） | 同縣市瓶配對 |
| 瓶子內容 | 您發送的訊息 | 提供服務功能 |
| 互動記錄 | 撈瓶、回覆、檢舉等 | 提供服務功能 |

### 我們不收集的資料

- 真實姓名
- 電子郵件
- 電話號碼
- 精確地理位置
- 其他可識別個人身份的資訊

## 2. 匿名機制

- 本服務採用 Supabase 匿名登入
- 無需提供任何個人資訊即可使用
- 您的匿名識別碼儲存在瀏覽器中
- 清除瀏覽器資料將無法恢復帳號

## 3. 資料使用

我們使用您的資料僅用於：

- 提供漂流瓶服務功能
- 處理檢舉和維護社群安全
- 改善服務品質

我們**不會**：

- 將您的資料出售給第三方
- 使用您的資料進行廣告投放
- 與其他服務共享您的資料

## 4. 資料儲存

- 資料儲存於 Supabase（AWS 基礎設施）
- 伺服器位於亞太地區（ap-northeast-1）
- 資料傳輸使用 HTTPS 加密

## 5. 資料保留

- 瓶子和回覆：用戶刪除後即移除
- 檢舉記錄：保留用於安全審查
- 帳號資料：長期不活躍可能自動清除

## 6. Cookie 使用

本服務使用必要的 Cookie 來：

- 維持您的登入狀態
- 儲存匿名識別碼

我們不使用追蹤型或廣告型 Cookie。

## 7. 您的權利

您可以：

- 刪除您發送的瓶子和回覆
- 清除瀏覽器資料以移除本地帳號
- 透過 GitHub Issues 請求刪除您的資料

## 8. 兒童隱私

本服務不針對 13 歲以下兒童。如果我們發現收集了兒童資料，將立即刪除。

## 9. 政策變更

隱私政策變更將在網站上公布。重大變更時，我們會在服務中通知用戶。

## 聯絡我們

如有隱私相關問題，請透過 GitHub Issues 聯繫我們。
